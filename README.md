**ITMO Stars Research Project** — программный комплекс для анализа ЭЭГ-активности и классификации ментального состояния оператора в режиме реального времени.

---

## 📋 Описание
Система предназначена для мониторинга функционального состояния операторов по данным электроэнцефалографии (ЭЭГ). Реализована в рамках научно-исследовательской работы Университета ИТМО.

**Актуальность:** Проект направлен на решение проблемы операторской усталости в критических системах управления, что повышает безопасность труда и эффективность мониторинга когнитивной нагрузки.

---

## 🏗 Архитектура

| Компонент | Технология | Назначение |
| :--- | :--- | :--- |
| **Backend** | FastAPI + Uvicorn | REST API для классификации и спектрального анализа |
| **Frontend** | Streamlit | Интерактивный веб-дашборд с визуализацией |
| **ML Core** | PyTorch + scikit-learn + XGBoost | EEG-2D-CNN + классические ансамбли моделей |
| **Explainability** | SHAP + LLM | Интерпретация решений модели и форматирование заключения |
| **Validation** | Subject-wise GroupKFold | Разделение по испытуемым без утечки данных |

## 🚀 Быстрый старт

### Вариант 1: Запуск через Docker Compose (Рекомендуемый) 🐳
Гарантирует развёртывание всей экосистемы (API + Dashboard) одной командой в изолированном окружении:

```bash
# 1. Клонирование репозитория
git clone https://github.com/fawk-ux/neuro-project.git
cd neuro-project

# 2. Запуск контейнеров в фоновом режиме
docker compose up -d --build
```

После старта сервисы доступны по адресам:
* 📊 **Streamlit Dashboard:** http://localhost:8501
* ⚡ **FastAPI Swagger Docs:** http://localhost:8000/docs

---

### Вариант 2: Локальный запуск (Без Docker) 💻

**Требования:** Python 3.10+

```bash
# Клонирование репозитория
git clone https://github.com/fawk-ux/neuro-project.git
cd neuro-project

# Создание и активация виртуального окружения
python -m venv .venv

# Windows (PowerShell / CMD)
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Быстрый старт на Windows
start.bat

# Или ручной запуск компонентов:
uvicorn api:app --host 0.0.0.0 --port 8000 --reload
streamlit run dashboard.py
```

## 📊 Возможности
1. **Визуализация ЭЭГ-сигналов:** Осциллограмма многоканальной записи, динамическая фильтрация, статистика признаков.
2. **Классификация состояния:**
    * **Relax / Neutral:** Стабильное бодрствование (нормализация α-ритма).
    * **Concentration:** Высокая концентрация (депрессия α, доминирование β).
    * **Mental Fatigue:** Умственное утомление / астенизация (дезорганизация ритмов, рост медленных волн).
3. **Спектральный анализ:** Расчёт спектральной мощности ритмов (δ, θ, α, β, γ) методом Уэлча, построение матриц распределения.

## 🧪 Датасет
Используется обработанный датасет **EEG brainwave dataset: mental state** *(Jordan J. Bird et al., IEEE 2018)*:
* 4 испытуемых, 3 состояния, 4 канала (нейроинтерфейс Muse).

---

## 🔬 Научная составляющая
* **EEG-2D-CNN:** Архитектура свёрточной нейросети для анализа пространственно-временной структуры ЭЭГ-сигналов.
* **Subject-wise GroupKFold:** Исключение data leakage за счёт кросс-валидации по новым испытуемым.
* **SHAP + Clinical Report:** Локальная интерпретация вкладов признаков и автоматическое формирование клинического резюме.
