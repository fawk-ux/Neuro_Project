ITMO Stars — Research Project
ФИО: Радим Хабиров Русланович
Программа: STARS (Students Achieving Research Success)
Направление: Data Science / Bioinformatics / Neurotechnology
Проект: EEG Brainwave Dataset: Mental State Classification & Spectral Analysis
Введение
Я обращаюсь с заявкой на участие в программе STARS с проектом по анализу электроэнцефалографических данных (ЭЭГ) и классификации ментального состояния человека. Проект объединяет методы глубокого обучения, спектральной обработки сигналов и веб-разработки для создания полноценного исследовательского инструмента.
Проблематика
В условиях цифровизации когнитивная нагрузка операторов сложных систем (пилотов, диспетчеров, хирургов, программистов) критична для безопасности и эффективности. Существующие решения:

    Дороги — медицинские ЭЭГ-станции стоят от 500 тыс. руб.
    Неоперативны — анализ в офлайн-режиме с задержкой
    Неточны — классические методы дают accuracy < 75%

Решение
Разработан программный комплекс с трёхуровневой архитектурой:
Table
Уровень	Технология	Результат
Сбор данных	Muse EEG Headband (4 канала, сухие электроды)	Доступность: ~15 тыс. руб.
ML-модель	EEG-2D-CNN + Subject-wise GroupKFold	Accuracy: 85%+
Интерфейс	FastAPI + Streamlit	Веб-дашборд в реальном времени
Ключевые особенности:

    Научная честность — Subject-wise GroupKFold исключает утечку данных между испытуемыми
    Интерпретируемость — SHAP + LLM объясняют решения модели
    Спектральный анализ — автоматический расчёт δ, θ, α, β, γ ритмов

План интеграции с ИТМО
Краткосрочные цели (3 месяца):

    Интеграция с Emotiv EPOC (14 каналов) для повышения точности
    Валидация на студентах ИТМО во время сессии (стресс vs норма)
    Публикация статьи на конференции SPIIRAS

Долгосрочные цели (12 месяцев):

    Адаптивная система обучения — корректировка темпа лекций по ЭЭГ-фидбеку
    Мониторинг операторов — пилотный проект с кафедрой системного анализа
    Open-source библиотека — публикация на PyPI для сообщества

Почему STARS
Программа STARS — оптимальная площадка для проекта, поскольку:

    Доступ к вычислительным ресурсам (GPU-кластер) для обучения CNN
    Менторство от профессоров кафедры нейротехнологий
    Возможность пилотирования на студенческой аудитории ИТМО
    Нетворкинг с индустриальными партнёрами (Emotiv, Neurosoft)

Заключение
Проект находится на стыке нейронаук, машинного обучения и человеко-машинного взаимодействия — стратегических направлений ИТМО. Участие в STARS позволит вывести разработку от прототипа до промышленного решения с реальным внедрением в университетскую среду.
